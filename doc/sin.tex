\subsection{sin}
\label{labsin}
\noindent Name: \textbf{sin}\\
the sine function.\\
\noindent Description: \begin{itemize}

\item \textbf{sin} is the usual sine function.

\item It is defined for every real number $x$.
\end{itemize}
